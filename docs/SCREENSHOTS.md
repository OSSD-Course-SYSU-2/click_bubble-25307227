# 游戏截图说明

由于这是一个开源项目，建议添加以下截图到 `docs/screenshots/` 目录：

## 需要的截图

1. **home.png** - 游戏主界面
   - 显示游戏标题、玩家信息、功能按钮

2. **game.png** - 游戏进行中界面
   - 显示泡泡、积分、时间、关卡信息

3. **level_select.png** - 关卡选择界面
   - 显示20个关卡的网格布局

4. **shop_items.png** - 道具商店界面
   - 显示5种道具及其价格

5. **shop_equipment.png** - 装备升级界面
   - 显示5种装备及其升级选项

6. **result.png** - 游戏结果弹窗
   - 显示通关/失败结果和奖励

## 截图要求

- 分辨率：1080x2340 或更高
- 格式：PNG
- 大小：每张不超过 2MB
- 内容：使用模拟器或真机截图

## 添加方式

1. 创建目录：`docs/screenshots/`
2. 将截图放入该目录
3. 在 README.md 中引用：
```markdown
![游戏主界面](docs/screenshots/home.png)
```
